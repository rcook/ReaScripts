# Tempo mapping

## Keyboard shortcuts

| Shortcut     | Description                                  | Type   |
| ---          | ---                                          | ---    |
| Ctrl+Shift+[ | Move edit cursor to start of current measure | Custom |
| Ctrl+Shift+] | Move edit cursor to start of next measure    | Custom |
| {            | Loop points: Set start point                 |        |
| }            | Loop points: Set end point                   |        |
